# แอปพลิเคชัน BMI Tracker

## ภาพรวม
ระบบการติดตามดัชนีมวลกาย (BMI) ที่สมบูรณ์ สร้างด้วยภาษา Java มีอินเตอร์เฟซผู้ใช้แบบกราฟิก (GUI) ที่ใช้งานง่าย แอปพลิเคชันนี้ช่วยให้ผู้ใช้บันทึกการวัดน้ำหนักและความสูง คำนวณ BMI และติดตามแนวโน้มเมื่อเวลาผ่านไป

## คุณสมบัติ

### 1. **ตรวจสอบความถูกต้องของข้อมูลนำเข้า**
   - ตรวจสอบน้ำหนัก (20-600 กิโลกรัม)
   - ตรวจสอบความสูง (50-300 เซนติเมตร)
   - ตรวจสอบความถูกต้องของตัวเลข
   - ข้อความข้อผิดพลาดที่เป็นมิตรต่อผู้ใช้

### 2. **ตรรมชาติของธุรกิจ**
   - **การคำนวณ BMI**: สูตร: BMI = น้ำหนัก (กก.) / (ความสูง (ม.))²
   - **การจำแนก BMI**:
     - น้ำหนักต่ำกว่าเกณฑ์: BMI < 18.5
     - ปกติ: 18.5 ≤ BMI < 25.0
     - น้ำหนักเกิน: 25.0 ≤ BMI < 30.0
     - อ้วน: BMI ≥ 30.0
   - **คำแนะนำเกี่ยวกับสุขภาพ**: ข้อเสนอแนะแบบบุคคลตามหมวดหมู่ BMI
   - **การวิเคราะห์แนวโน้ม**: ติดตามแนวโน้ม BMI

### 3. **การจัดเก็บและเรียกข้อมูล**
   - การจัดเก็บแบบถาวรโดยใช้ Java Serialization
   - บันทึกข้อมูลไปยังไฟล์ `bmi_data.dat`
   - โหลดบันทึกที่เก่าโดยอัตโนมัติเมื่อเริ่มต้น
   - การดำเนินการ CRUD แบบสมบูรณ์ (สร้าง อ่าน อัปเดต ลบ)

### 4. **แสดงผลข้อมูล**
   - **มุมมองแบบแท็บ**: 5 หน้าแยก (ยินดีต้อนรับ ป้อนรายละเอียด ผลลัพธ์ แผนภูมิ ประวัติ)
   - **กราฟแสดง BMI**: แผนภูมิเส้นแสดง BMI เมื่อเวลาผ่านไป
   - **ประวัติ**: รายการบันทึกทั้งหมดพร้อมตัวเลือกแก้ไขและลบ

## แท็บแอปพลิเคชัน GUI

### 1. แท็บ "ยินดีต้อนรับ" (Welcome)
- หน้าแรกของแอปพลิเคชัน
- ปุ่ม "View History" - ดูประวัติบันทึก
- ปุ่ม "Start Tracking" - เริ่มการบันทึกใหม่

### 2. แท็บ "ป้อนรายละเอียด" (Enter Details)
- ช่องป้อนน้ำหนัก (หน่วย: กิโลกรัม)
- ช่องป้อนความสูง (หน่วย: เซนติเมตร)
- ปุ่ม "Reset" - ล้างข้อมูล
- ปุ่ม "Calculate BMI" - คำนวณ BMI

### 3. แท็บ "ผลลัพธ์ BMI" (BMI Results)
- แสดง BMI ค่า (สีหนา, ตัวอักษรใหญ่)
- แสดงสถานะน้ำหนัก (Normal, Overweight, etc.)
- แสดงการเปลี่ยนแปลง BMI (↑ ↓ หรือ "ไม่มีการเปลี่ยนแปลง")
- ปุ่ม "Share Results" - คัดลอกไปยังคลิปบอร์ด
- ปุ่ม "Save Record" - บันทึกลงฐานข้อมูล

### 4. แท็บ "แผนภูมิ" (BMI Chart)
- แสดงกราฟแนวโน้ม BMI
- พล็อตจุดข้อมูลและเส้นเชื่อม
- แสดง min/max/range ของ BMI
- อัปเดตโดยอัตโนมัติเมื่อเพิ่มบันทึกใหม่

### 5. แท็บ "ประวัติ" (History)
- แสดงรายการบันทึกทั้งหมด (วันที่, BMI, หมวดหมู่)
- สามารถเลือกบันทึก (คลิกเพื่อเลือก)
- ปุ่ม "Delete Entry" - ลบบันทึกที่เลือก
- ปุ่ม "Edit Entry" - แก้ไขบันทึกที่เลือก
- อัปเดตโดยอัตโนมัติหลังจากบันทึก/ลบ

## หมวดหมู่ BMI และคำแนะนำ

| หมวดหมู่ | ช่วง BMI | คำแนะนำ |
|-----------|-----------|---------|
| น้ำหนักต่ำกว่าเกณฑ์ | < 18.5 | พยายามเพิ่มน้ำหนักผ่านการบริหารอาหารและออกกำลังกาย |
| ปกติ | 18.5 - 24.9 | รักษาน้ำหนักปัจจุบันของคุณผ่านการใช้ชีวิตที่ดี |
| น้ำหนักเกิน | 25.0 - 29.9 | พิจารณาลดน้ำหนักผ่านการกำหนดอาหารและออกกำลังกาย |
| อ้วน | ≥ 30.0 | ปรึกษาผู้เชี่ยวชาญด้านสุขภาพสำหรับแผนการจัดการน้ำหนัก |

## การจัดเก็บข้อมูล

- บันทึกทั้งหมดจะถูกบันทึกโดยอัตโนมัติไปยังไฟล์ `bmi_data.dat`
- ใช้ Java Object Serialization (ไม่มี SQL database)
- ข้อมูลยังคงอยู่ระหว่างการใช้งานแอปพลิเคชัน
- บันทึกรวมลงวลันเวลาเพื่อติดตามการเปลี่ยนแปลง

## คำอธิบายคลาส

### BMIRecord.java
- โมเดลข้อมูลแสดงการวัด BMI ครั้งเดียว
- เก็บ: น้ำหนัก, ความสูง, BMI, หมวดหมู่, ลงวลันเวลา
- คำนวณ BMI โดยอัตโนมัติเมื่อสร้าง
- ใช้งาน Serializable เพื่อความถาวรของข้อมูล

### BMITrackerGUI.java
- แอปพลิเคชันอินเตอร์เฟซผู้ใช้กราฟิกแบบหลายแท็บ
- 5 แท็บแยก: ยินดีต้อนรับ, ป้อนรายละเอียด, ผลลัพธ์, แผนภูมิ, ประวัติ
- จัดการการนำทาง, การแสดงผล, และการเชื่อมต่อกับตัวจัดการข้อมูล
- รองรับการแก้ไขและลบรายการประวัติ

### BMIDataManager.java
- จัดการการจัดเก็บและเรียกข้อมูล BMI
- ดำเนินการจัดการไฟล์ I/O
- ใช้งานการโหลดและบันทึกข้อมูล
- ให้วิธีการเพิ่ม, เรียกข้อมูล, และลบบันทึก

### BMICalculator.java
- ประกอบด้วยตรรมชาติของธุรกิจสำหรับการคำนวณ BMI
- คำนวณการจำแนก BMI
- คำนวณ BMI เฉลี่ยและแนวโน้ม
- สร้างคำแนะนำเกี่ยวกับสุขภาพ

### InputValidator.java
- ตรวจสอบอินพุตผู้ใช้ทั้งหมด
- ตรวจสอบค่าตัวเลข
- ตรวจสอบช่วงน้ำหนักและความสูง
- ให้ข้อความข้อผิดพลาดการตรวจสอบ

## รายละเอียดทางเทคนิค

- **ภาษา**: Java
- **กรอบการทำงาน**: ไม่มี (Java บริสุทธิ์)
- **GUI Framework**: Java Swing
- **รูปแบบข้อมูล**: Java Object Serialization
- **ไฟล์ข้อมูล**: bmi_data.dat (ไบนารี)
- **Database**: ไม่มี (ไฟล์ Object Serialization)

## หมายเหตุ

- ลงวลันเวลาทั้งหมดจะถูกบันทึกในเวลาระบบในประเทศ
- ค่า BMI จะถูกปัดเศษเป็นทศนิยม 1 ตำแหน่งเพื่อความแม่นยำ
- แอปพลิเคชันสร้างไฟล์ `bmi_data.dat` ในไดเรกทอรีการทำงาน
- ไม่จำเป็นต้องมีการอ้างอิงภายนอก - ใช้เฉพาะ Java Standard Library

## คำปฏิเสธ

ตัวติดตาม BMI นี้มีไว้เพื่อวัตถุประสงค์ข้อมูล ไม่ควรใช้แทนคำแนะนำทางการแพทย์ขององค์กรวิชาชีพ โปรดปรึกษาผู้เชี่ยวชาญด้านสุขภาพเพื่อรับคำแนะนำส่วนตัว

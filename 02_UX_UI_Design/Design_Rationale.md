Design Rationale

เหตุผลในการออกแบบสำหรับ "BMI Tracker"

เอกสารนี้อธิบายเหตุผลเบื้องหลังการออกแบบ User Experience (UX) และ User Interface (UI) ของระบบ BMI Tracker โดยเน้นที่ประสิทธิภาพ และความเข้าใจง่ายของผู้ใช้เป็นหลัก

1\. วัตถุประสงค์หลักของ UX/UI

วัตถุประสงค์หลักของการออกแบบคือการทำให้ผู้ใช้สามารถ **ป้อนข้อมูล (น้ำหนัก/ส่วนสูง)**, **ดูผลลัพธ์ (BMI)**, และ **ติดตามประวัติย้อนหลัง** ได้ภายในหน้าจอเดียว **โดยมีขั้นตอนน้อยที่สุด (Minimal Steps)และผู้ใช้เข้าใจได้ง่าย**

2.เหตุผลในการออกแบบ Layout และ Flow

การจัดวางองค์ประกอบหลักทั้งหมด ไว้ในหน้าจอเดียวทำให้ผู้ใช้ไม่จำเป็นต้องคลิกหลายหน้าเพื่อดูข้อมูลสำคัญ

3.เหตุผลในการเลือกใช้สีและองค์ประกอบภาพ

สร้างความรู้สึก **สะอาด**, **เรียบง่าย**  และ **เป็นมืออาชีพ** เหมาะสำหรับแอปพลิเคชันด้านสุขภาพ/ข้อมูล
